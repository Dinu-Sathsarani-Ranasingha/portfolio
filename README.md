## About

This portfolio highlights my background in business analysis, project management, and data analytics, along with featured projects, certifications, education, and experience. It's built as a single self-contained `index.html` file — no build tools or frameworks required.

## Features

- Dark navy/purple themed UI with a glassy hero section and animated particle background
- Responsive layout with mobile hamburger navigation
- Featured project cards with GitHub links and a YouTube video modal
- Skills, Education, Certifications, and Experience sections
- Contact section with direct links (Email, LinkedIn, GitHub)
- Smooth scroll-reveal animations on section elements

## Tech Stack

- **HTML5** — semantic single-page structure
- **CSS3** — custom properties (CSS variables), Flexbox/Grid layouts, responsive design
- **Vanilla JavaScript** — scroll reveal, particle canvas animation, mobile nav toggle, video modal
- **Google Fonts** — Poppins
- **Font Awesome 6** — icon set (via CDN)

## Running Locally

Since this is a static single-file site, no build step is needed:

```bash
git clone https://github.com/Dinu-Sathsarani-Ranasingha/<repo-name>.git
cd <repo-name>
```

Then simply open `index.html` in your browser, or serve it locally:

```bash
python3 -m http.server 8000
```

Visit `http://localhost:8000` in your browser.

## Deployment

This site can be deployed on any static hosting provider, such as:

- **GitHub Pages** — enable Pages in repository settings, pointing to the `main` branch
- **Netlify** or **Vercel** — connect the repository and deploy with default static site settings
